## Usage Guidelines

- Use **Primary Green** for primary buttons, active states, and interactive elements.
- Use **Dark Green** for headings and body text where strong contrast is needed.
- Use **Light Green (#EDF6F2)** for hover states and subtle selection backgrounds.
- Reserve **Error** and **Warning** for validation messages and destructive actions only.
- Never use raw hex values in component code — use MUI theme tokens or CSS variables.
