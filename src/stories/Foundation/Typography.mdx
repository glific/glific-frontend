# Typography

Glific uses **Heebo** as its primary typeface, applied globally via the MUI theme at `src/config/theme.tsx`.

```ts
// src/config/theme.tsx
typography: {
  fontFamily: ['heebo', 'sans-serif'].join(','),
}
```

## Tabs & Compact UI

Tabs have a fixed height of `32px` (overridden in theme) to keep the top navigation compact:

```ts
MuiTabs: { styleOverrides: { root: { height: '32px', minHeight: '32px' } } }
MuiTab:  { styleOverrides: { root: { height: '32px', minHeight: '32px' } } }
```

---

## Usage Guidelines

- Import the `Typography` component from MUI to use semantic variants.
- Never hardcode `font-family: heebo` in component CSS — the body-level override in `index.css` and MUI theme handle it globally.
- Use `variant="caption"` for `Input` field labels (`inputLabel` prop).
- Use `variant="h5"` for `Dropdown` field labels.
